# AllCare Clinic Scheduler

## Overview

The AllCare Clinic Scheduler is a web-based appointment scheduling system developed to improve the management of clinic appointments, patient records, and doctor availability. The system provides an efficient way for patients and clinic staff to organize appointments while reducing scheduling conflicts and administrative workload.

## Features

* Patient registration and management
* Appointment scheduling
* Appointment cancellation and rescheduling
* Doctor availability tracking
* Search functionality
* Responsive user interface

## Problem Statement

Many clinics rely on manual appointment booking processes, which can lead to double bookings, scheduling conflicts, and inefficiencies. The AllCare Clinic Scheduler was developed to provide a streamlined and user-friendly solution for managing appointments digitally.

## Technologies Used

* HTML5
* CSS3
* JavaScript
